Slow-Motion Millisecond Timer

Arduino-based timing tool for analysing high-speed events captured in slow-motion video.

Overview

This project is a compact Arduino-based timer designed for analysing high-speed events in slow-motion video. It displays time in centiseconds using a TM1637 4-digit display.

Indoors the display records perfectly. Outdoors, the camera shutter becomes extremely fast, causing each digit's refresh cycle to appear separately — which creates flickering or ghosting in the slow-mo footage.

My contributions

  • Selected and wired the Arduino, TM1637, button, batteries and switch.
  • Wrote the Arduino timing and display logic.
  • Installed components in a compact enclosure.
  • Used the timer extensively on motor spool-up analysis for ENDURA.

Parts list

  • Arduino Pro Mini
  • TM1637 4-digit display
  • Momentary push button
  • 1 kΩ resistor
  • 2× AAA batteries and switch
  • Small project enclosure
