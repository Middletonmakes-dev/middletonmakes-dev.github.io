Slow-Motion Millisecond Timer

Arduino-based timing tool for analysing high-speed events captured in slow-motion video.

Overview

This project is a compact Arduino-based timer designed specifically for analysing high-speed events captured in slow-motion video. The timer continuously displays elapsed time in centiseconds (1/100th of a second) using a bright 4-digit numerical display. By placing the timer in frame while recording slow-motion footage, you can accurately measure how long an object takes to travel between two points, accelerate, or complete any fast motion that’s too quick for ordinary timing tools.

A simple button interface allows you to start, stop, and reset the timer. An unfortunate oversight was that, indoors, the timer displays perfectly in slow-motion footage, but in direct sunlight the digits became harder to read. This is caused by a shutter-speed mismatch between the camera and the TM1637 display.

The display refreshes each digit in rapid sequence, which is normally invisible to the eye. However, in bright outdoor light the camera uses a much faster shutter speed, capturing individual refresh cycles. This creates a cascading or flickering effect in slow-motion video, making the digits less clear.

This timer was used extensively with the ENDURA project while getting motor spool-up tuned for the launcher (see the UAV launcher rail) and provided a simple, repeatable way to measure timing between key frames in slow-motion footage.

My contributions

  • Selected and wired the Arduino Pro Mini, TM1637 display, push button, power switch, and battery pack.
  • Developed the Arduino code to handle button input, timing logic, and centisecond display on the TM1637.
  • Assembled the complete unit in an off-the-shelf enclosure with a simple field-friendly layout.
  • Used the timer in the field alongside slow-motion video to measure motor spool-up and launch timing.
  • Reviewed the recorded footage frame-by-frame and used the timing data to refine flight controller settings.
  • Documented the behaviour of the display under different lighting conditions (indoor vs direct sunlight) and the shutter-speed interaction.

Parts list

  • Arduino Pro Mini
  • TM1637 4-digit 7-segment display (0.56", red)
  • Momentary push button
  • 1 kΩ resistor
  • Power source – 2× AAA batteries
  • On/off toggle switch
  • Off-the-shelf project box / enclosure
  • Basic hookup wire and headers
